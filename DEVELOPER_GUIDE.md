# 📘 WealthFlow Developer Setup, Configuration & End-User Manual

This document provides setup guides, project configuration files, runtime rules, and user guides for **WealthFlow**.

---

## 1. Local Developer Workspace Setup

Follow these steps to set up and run the Android mobile client and the Django REST Framework API backend locally.

### Prerequisites
Ensure your local development machine has the following tools installed:
- **Java SE Development Kit (JDK) 17** (Used to build the Kotlin Android target)
- **Android Studio (Hedgehog 2023.3+ or Ladybug 2024.1+)**
- **Python 3.11+** with virtualenv package managers
- **Docker Engine 24+** with **Docker Compose**
- **PostgreSQL 15** and **Redis Server 7**

---

## 2. API Backend Local Setup

1. **Clone the Repository & Navigate to Backend**:
   ```bash
   git clone https://github.com/wealthflow/platform.git
   cd platform/backend
   ```

2. **Initialize Python Virtual Environment**:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

3. **Install Core Package Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Configure Environment Variables (`.env`)**:
   Create a local configuration file `.env` inside `/backend/wealthflow/` with these values:
   ```env
   DJANGO_SECRET_KEY="production-quality-crypto-secret-key"
   DEBUG=True
   DATABASE_URL="postgres://postgres:postgres@localhost:5432/wealthflow_dev"
   REDIS_URL="redis://127.0.0.1:6379/1"
   CELERY_BROKER_URL="redis://127.0.0.1:6379/0"
   GOOGLE_CLIENT_ID="your-google-oauth-client-id-here"
   GEMINI_API_KEY="your-google-gemini-api-token"
   STRIPE_SECRET_KEY="sk_test_..."
   ```

5. **Run Local Database Migrations**:
   ```bash
   python manage.py migrate
   ```

6. **Start Django DRF Server**:
   ```bash
   python manage.py runserver 0.0.0.0:8000
   ```

7. **Start Background Celery Workers**:
   Open a separate shell terminal and launch the worker instances:
   ```bash
   celery -A wealthflow worker --loglevel=info
   ```

---

## 3. Android Mobile Application Setup

1. **Launch Android Studio** and select **Open existing project**. Choose `/platform/app` from your directory tree.
2. **Synchronize Gradle Files**: Allow Android Studio to resolve the dependencies listed in `/gradle/libs.versions.toml`.
3. **Configure Local Environment variables (`.env`)**:
   Create a local configuration `.env` file inside the root directory to match the following keys:
   ```env
   GEMINI_API_KEY="your-google-gemini-api-token"
   BASE_URL="http://10.0.2.2:8000" # Maps local emulator to localhost API ports
   ```
4. **Compile and Execute**: Click **Run App** (green play button) or use terminal command lines:
   ```bash
   gradle :app:assembleDebug
   ```

## 4. Platform End-User Feature Manual

WealthFlow provides a polished, intuitive interface designed to help users take control of their finances.

### 1. Unified Ledger & Account Syncing
- **Connecting Accounts**: Navigate to **Accounts** inside the main dashboard, select **Add Account**, and choose between Cash, Checking, Savings, or Credit Cards.
- **Entering Cash Flows**: Tap the floating action button (FAB) `+` to log physical expenditures. Enter the amount, select a category, and optionally snap a photo of the receipt to trigger auto-categorization.
- **Exporting Records**: Tap **Export** on the Ledger screen to generate a formatted CSV statement of your finances, ready for import into spreadsheet applications.

### 2. Interactive Category Budget Tracking
- **Setting Budgets**: Navigate to the **Budgets** tab. Choose a category (e.g. Food, Travel, Utilities) and set a monthly limit.
- **Dynamic Spend Meters**: Color-coded progress bars show your remaining budget in real-time. If expenses exceed $80\%$ of your category budget, you'll receive a proactive push notification.

### 3. Copilot AI Financial Advisor
- **Initiating Chat**: Open the **Copilot** tab to speak with your AI Financial Advisor.
- **Natural-Language Audits**: Ask the AI complex questions about your finances, such as:
  - *"How much did I spend on dining out last month?"*
  - *"Can you optimize my active budgets to help me save $500?"*
  - *"Analyze my recurring subscriptions for price increases or duplicate charges."*
- **Actionable Advice**: Tap any recommendation generated by the AI to automatically update a budget or cancel a duplicate subscription.

### 4. Smart Liabilities & Debt Reduction
- **Debt Tracking**: Navigate to the **Bills & Debt** tab. Add your credit cards and loans with their respective interest rates (APR).
- **Payment Calculators**: Toggle between **Avalanche** (highest interest first) and **Snowball** (smallest balance first) repayment strategies. The app dynamically calculates your projected debt-free date and total interest savings for each method.
